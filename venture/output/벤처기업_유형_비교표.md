## 13. 벤처기업 vs 소셜벤처기업 비교

> 소셜벤처기업은 벤처특별법 **별도 조항**(제16조의10)에 따른 **판별 제도**이며, 벤처기업 **확인**과는 신청·심사·결과 문서가 다르다. 두 제도는 **병행 취득 가능**하며, 벤처기업 확인 보유 시 소셜벤처 혁신성장성 판별에서 가점(100점)을 받을 수 있다.

| 구분 | 벤처기업 (확인제도) | 소셜벤처기업 (판별제도) |
|------|:------------------:|:---------------------:|
| **법적 근거** | 벤처특별법 제2조의2, 제25조의4 | 벤처특별법 제2조⑩, 제16조의10 |
| **운영 요령** | [벤처기업확인요령](https://www.law.go.kr/행정규칙/벤처기업확인요령) | [소셜벤처기업 지원제도 운영요령](https://dams.pa.go.kr/dams/DOCUMENT/2023/11/30/DOC/SRC/0A04202311301640183855019015421.pdf) |
| **핵심 정의** | 기술 혁신성·사업 성장성 우수 중소기업 | 사회적 가치·경제적 가치 **통합** 추구 기업 |
| **유형/분류** | 4가지 확인 유형 (단일 선택) | 사회성 + 혁신성장성 (이중 평가) |
| **합격 기준** | 유형별 정량·정성 요건 + 위원회 의결 | 사회성 **70점+** AND 혁신성장성 **70점+** |
| **신청 자격** | 중소기업 (예비벤처는 등록 준비 중 포함) | **영리법인** (개인사업자·비영리 불가) |
| **주관 기관** | (사)벤처기업협회 (확인기관) | 기술보증기금 (판별기관) |
| **신청 플랫폼** | [벤처인](https://www.smes.go.kr/venturein) | [소셜벤처스퀘어](https://sv.kibo.or.kr/) |
| **결과 문서** | 벤처기업확인서 | 소셜벤처기업 판별 결과 통지서 |
| **유효기간** | 3년 (재확인 필요) | 별도 유효기간 없음 (판별 시점 기준) |
| **처리 기한** | 30~45일 | 21일 이내 |
| **수수료** | 275,000~605,000원 (유형별) | 운영요령상 별도 규정 (판별기관 안내) |
| **자가진단** | 벤처유형 자가진단 (권장) | **필수** — 각 60점+ 시 신청 가능 |
| **현장실사** | 유형별 상이 | 판별기관 재량 (서류심사 중심) |
| **주요 혜택** | 세제·금융·입지·인력·조달 우대 | 기술보증·투자, 육성사업, 실태조사 기반 지원 |
| **2025년 규모** | 벤처확인 38,598개사 ([벤처인 홈](https://www.smes.go.kr/venturein/home/viewHome)) | 실태조사 모집단 약 2,679개사 (2023) |

### 13.1 소셜벤처 판별 절차

```
1. 소셜벤처스퀘어 회원가입
2. 자가진단 (사회성·혁신성장성 각 60점+)
3. 신청서·사업요약서·증빙서류 제출
4. 판별기관(기보) 서류심사 + 민간자문위원회 심의
5. 판별 결과 통지서 발급 (21일 이내)
6. 불복 시 7일 이내 이의제기 → 재심사
```

### 13.2 벤처기업 확인이 소셜벤처에 미치는 영향

| 벤처기업 상태 | 소셜벤처 혁신성장성 영향 |
|--------------|------------------------|
| 벤처기업·이노비즈·메인비즈 **인증 보유** | 혁신성장성 판별표 1항 **100점** (즉시 합격 요건 충족에 유리) |
| 벤처투자 50백만원+ 유치 | 혁신성장성 5항 **100점** |
| 기업부설연구소 보유 | 혁신성장성 10항 **30점** |
| 미확인 | 기술평가·IP·성장률·R&D비 등 다른 항목으로 70점 충족 필요 |

### 13.3 소셜벤처 사회성 고득점 경로

| 경로 | 점수 |
|------|:----:|
| 사회적기업·협동조합·B-corp 등 인증 | 100 |
| 소셜임팩트 투자 유치 | 100 |
| 사회적 문제 해결 제품·서비스 사업화 | 70 |
| 정관에 사회적 가치·이윤배분·이해관계자 참여 명시 | 30~50 |

> 출처: [운영요령 별표2](https://dams.pa.go.kr/dams/DOCUMENT/2023/11/30/DOC/SRC/0A04202311301640183855019015421.pdf), [소셜벤처 실태조사 안내](https://mods.go.kr/boardDownload.es?bid=12013&list_no=437453&seq=1)
