# DeepCube.AI 제품 소개(요약)

탈 RAG · DB 직결형 Agentic AI Platform. 사내 문서(RAG) 뿐 아니라 기업의 실시간 데이터(기존 RDB 연동)를 한 시나리오에서 동시 처리가능한 엔터프라이즈 AI 플랫폼.


## 핵심 기능
- 시각적 AI 워크플로우(시나리오): 노코드 GUI로 누구나 설계·테스트 가능하고, 배포·운영까지 한 플랫폼에서 처리
- 문서 Q&A(RAG): 사내 정책·매뉴얼·보고서 벡터 검색 기반 답변
- DB 실시간 조회: Text-to-SQL로 자연어 → SQL 자동 변환, Oracle·MySQL·PostgreSQL 등 주요 RDBMS 직결
- 복합 자동화: 문서 확인 → DB 조회 → 응답 생성 등 다단계 업무를 GUI에서 노드로 연결
- 모듈형 AI: 다중 LLM(기업 자체 LLM·OpenAI·Anthropic·Upstage·Google 지원), 웹 검색, 필요 역량만 조합
- 기업형 운영: RBAC·부서별 권한 분리·감사 로그·승인
- 시스템 연동: 표준 API로 포털·백오피스·앱에 AI 삽입 가능
- 배포·보안: 온프레미스·로컬 LLM 지원, 데이터 외부 유출 차단, 배포 승인

## 특장점
1. 현업이 직접 설계 — 개발 의존 없이 드래그 앤 드롭·실시간 테스트
2. 문서+DB 동시 지원 — RAG 뿐 아니라 RDB도 조회 => 정적인 문서 뿐 아니라 실시간 현황 처리 가능
3. 조직 통제 — 권한·시나리오·실행 이력을 운영 콘솔에서 중앙 관리
4. 기존 시스템 연동 — 기존 RDB 이용 뿐 아니라, 기존 웹 포탈, 그룹웨어 등에서 API 호출하여 AI 기능 추가 및 이용 가능
5. 기업 환경 대응 — 온프레미스·인증·감사 체계로 망·데이터 정책 준수
